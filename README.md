#  📊  Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using transactional data from 3,900 purchases across various product categories with  **Python, SQL, and Power BI**. The goal is to uncover insights into spending patterns, customer segments, product preferences, and subscription behavior to guide strategic business decisions. It covers the complete pipeline—from data cleaning and exploration to business analysis and interactive visualization — aimed at extracting meaningful insights into customer behavior.


##  📂 Dataset Summary
Rows: 3,900 
Columns: 18 
Key Features: - Customer demographics (Age, Gender, Location, Subscription Status) - Purchase details (Item Purchased, Category, Purchase Amount, Season, Size, Color) - Shopping behavior (Discount Applied, Promo Code Used, Previous Purchases, Frequency of Purchases, Review Rating, Shipping Type) 
Missing Data: 37 values in Review Rating column 


## 🚀 Key Components

### 🔹 Data Preparation & Exploratory Data Analysis (Python)

Used **Python** to clean, preprocess, and transform raw data into an analysis-ready format. Performed exploratory data analysis (EDA) to identify patterns, trends, and potential anomalies.



### 🔹 Data Analysis (SQL)

Utilized **SQL** to simulate business scenarios and execute analytical queries. Extracted insights related to customer segmentation, loyalty patterns, and key purchase drivers.


### 🔹 Visualization & Insights (Power BI)

Developed an **interactive Power BI dashboard** to present findings in a clear and intuitive way. The dashboard highlights important trends and supports data-driven decision-making.
